# Google Search Console Submission Checklist

Complete only after local QA, owner approval, and deployment.

- [ ] Confirm `https://dothiscoach.com/` is verified in Google Search Console.
- [ ] Open the deployed `robots.txt` and confirm crawling is allowed.
- [ ] Open the deployed `sitemap.xml` and confirm all 10 article URLs return HTTP 200.
- [ ] Submit `https://dothiscoach.com/sitemap.xml` in Search Console.
- [ ] Inspect `/articles/` and one article URL with URL Inspection.
- [ ] Request indexing for the article index and the highest-priority beginner guide.
- [ ] Validate `Article` and `BreadcrumbList` markup with Google's Rich Results Test.
- [ ] Check Page Indexing after one to two weeks for canonical or crawl errors.
- [ ] Review Search Results performance by page and query monthly.
- [ ] Update weak titles or intros only when impressions show a real intent mismatch.
- [ ] Do not submit thin keyword variants or request indexing repeatedly.
